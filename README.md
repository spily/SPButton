# SPButton
一行代码自定义按钮图文间距、通过runtime方式自适应，`**不需要继承**`，`**高度解藕**`、`**即插即用**`

# 如何使用
- 导入button的分类，给button赋能
```
  #import "UIButton+ContentLayout.h"
  ```
- 设置图文位置和间距
```
  [self.button setImageLayout:UIButtonLayoutImageLeft space:8];
  ```
